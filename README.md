# 환불 지킴이 (Refund Guardian)

**포기하기 전, 내 환불 권리와 다음 행동을 정확히 확인한다.**

헬스장·필라테스 선결제 먹튀(피해구제 3년 14,857건), 온라인쇼핑 환불 거부, 구독 해지 방해.
법이 보장한 권리(청약철회·중도해지·할부항변권)가 있는데 몰라서 포기하는 사람이 대부분입니다.
환불 지킴이는 상황을 대화로 말하면 **적용 법령과 기한, 지금 할 일**을 짚어주는 카카오톡 안 소비자 권리 비서입니다.

## 툴 4종 (무인증, 외부 API 불필요)
| 툴 | 역할 |
|---|---|
| `check_refund_right` | 상황 서술 → 적용 법령(청약철회 7일/14일·중도해지·항변권)·요건·기한·다음 단계 |
| `check_installment_defense` | 할부항변권 수량 요건·6개 법정 사유 후보 점검 후 카드사 공식 확인 경로 안내 |
| `generate_refund_letter` | 판매자 대상 내용증명 초안 생성·카드 분쟁은 공식 카드사 양식과 증빙 확인 경로 안내 |
| `refund_channels` | 1372·소비자원·홈택스 폐업조회·금감원·소액심판 안내 |

카드사 지급거절 행사문은 자동 생성하지 않습니다. `ground_code`, `ground_confirmed`와
필수 `ground_facts`는 법정 사유 후보를 빠짐없이 점검하는 데 쓰지만, 같은 대화 입력만으로는
증빙 진위·동일 계약 관련성·법 적용을 독립 확인할 수 없기 때문입니다. 수량 요건과 사유가
충족해 보여도 `review`로 보내 카드사 공식 양식과 증빙 확인, 1332·1372 경로를 안내합니다.
판매자 대상 환불 요구서 초안 기능은 그대로 제공합니다.

## 실행
```bash
pip install -r requirements.txt
python server.py   # Streamable HTTP, {HOST:0.0.0.0, PORT:8000}/mcp
```

- 안내는 법령·소비자분쟁해결기준 근거 참고 정보(법률 자문 아님), 최종 확인은 1372.
- 개인정보 비저장: 입력은 요청 처리 후 즉시 폐기.
